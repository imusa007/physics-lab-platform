\section*{Lab 1: Constant Acceleration}

In this lab you will measure the acceleration of a cart on a track.

\subsection*{Objective}
Determine the acceleration $a$ using position vs. time data.

\subsection*{Theory}
The kinematic equation for constant acceleration is:
\[ x(t) = x_0 + v_0 t + \frac{1}{2} a t^2 \]

\subsection*{Procedure}
1. Record position measurements at equal time intervals.\\
2. Enter your $(t, x)$ data in the web interface.\\
3. Use the generated plot to estimate the acceleration.

\subsection*{Analysis}
Use the data you collected to compute acceleration.
